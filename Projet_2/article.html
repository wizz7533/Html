CSS particulier à une page et Animations CSS

IL est possible de réutiliser la quasi-totalité d'un CSS existant pour une page avec un affichage de contenu légèrement différent, par exemple cette page qui affiche un seul article utilise un CSS supplémentaire qui ne surcharge que quelques règles pour adapter le style de l'article.

Animations en CSS

Pour utiliser les transitions il faut tout d'abord spécifier sur une classe la propriété transition, il faut définir dans cette propriété les différentes propriétés CSS qui utiliseront une transition. Ensuite par exemple sur la pseudo classe hover liée à la classe précédente, je décris les styles des transformations.

transition, scale

Scale() sert à augmenter la taille à l'échèle d'un élément, cela augmente tout, c'est à dire l'élément et son contenu.

Transition, rotate

Rotate fait effectuer à l'élément une rotation, on peut choisir le point pivot de l'élément.

Transition, Translate

Translate fait effectuer à l'élément une translation dans une direction donnée.

Animation, animate

Animate permet de créer des animations sur des éléments. On peut définir des images clefs (keyframe) sur un pourcentage de la durée totale de l'animation, à chaque keyframe on définir l'état ou les style de l'élément à cette keyframe.
