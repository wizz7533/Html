Différencier les feuilles de style

Il est possible d'associer plusieurs feuilles de style à une page html, et de différencier les "media" (screen, print) qui les utiliseront.

Si une page utilise 90% du CSS de base mais nécessite 10% de changement, comme l'affichage d'un article unique, il est souvent créé un feuillet CSS qui surchargera les style uniquement sur ce type de page

Inclure le responsive

Il est maintenant généralisé le fait d'écrire son CSS responsive en mobile first, car la majorité des personne allant sur internet le font à l'aide d'un smartphone ou d'une tablette.

De plus il est plus simple de le faire de cette manière, beaucoup moins de CSS à réécrire selon les tailles d'affichage.
